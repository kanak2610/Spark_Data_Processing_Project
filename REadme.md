# 📊 Sales Analytics Using Apache Spark

## 🚀 Project Overview

This project demonstrates large-scale sales data analysis using Apache Spark (PySpark). The dataset contains transaction records from multiple regions, cities, stores, and product categories. The objective is to process big data efficiently and generate meaningful business insights through Spark transformations, aggregations, and SQL queries.

---

## 🎯 Project Objectives

* Load and process large datasets using Apache Spark.
* Clean and transform raw transaction data.
* Perform sales and revenue analysis.
* Analyze customer and subscription trends.
* Generate business insights using Spark SQL.
* Demonstrate distributed data processing capabilities.

---

## 🛠️ Technologies Used

* Apache Spark (PySpark)
* Python
* Jupyter Notebook
* CSV Dataset

---

## 📂 Dataset Features

The dataset includes:

* User ID
* Transaction Date
* Region
* City
* Product Category
* Sale Amount
* Price

* Subscription Type
* Store ID
* Status

Total Records: **5000+**

### 3️⃣ Data Transformation

* Create derived columns

### 4️⃣ Exploratory Analysis

### 5️⃣ Aggregations

### 6️⃣ Spark SQL Analytics

* Execute analytical queries
* Generate business reports
